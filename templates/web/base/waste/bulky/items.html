Important information

Maximum numbers

The following types of item have a maximum number that can be collected in one collection:

  • Batteries: 1 tied bag of domestic batteries (min 10 - max 100)
  • Coffee pods: Up to 3 - each one must be in a Podback bag
  • Paint: Up to 4 - each one up to 5L (eg 1 x 5 litre tin, 5 x 1 litre tins)
  • Textiles: Up to 3 - each one up to 60L (eg 3 carrier bags or one black bin bag)
  • Small WEEE: Up to 4 small WEEE items
[% END %] [% IF c.cobrand.moniker == 'brent' %]

Important information

Maximum types

We can only collect 3 types of item in one collection

So you can have a collection of batteries, an iron, a kettle and a toaster, and two lots of textiles. But if you wanted to include paint, you would need to remove one of the following:

  • the batteries
  • the iron, the kettle and the toaster
  • all the textiles

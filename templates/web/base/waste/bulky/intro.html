Before you start your booking

  1. Requesting a bulky waste collection usually takes approximately 10 minutes
  2. You can request up to [% c.cobrand.bulky_items_maximum | numwords %] items per collection
    3. [% IF cobrand.moniker == 'kingston' || cobrand.moniker == 'sutton' %] [% USE pounds = format('%.2f'); SET cfg = c.cobrand.wasteworks_config; SET band1_items = cfg.band1_max; SET max_items = cfg.items_per_collection_max; SET base_price = cfg.base_price / 100; SET band1_price = cfg.band1_price / 100; %]
  3. The price you pay depends how many items you would like collected:
    • Up to [% band1_items %] items = £[% pounds(band1_price) %]
    • Up to [% max_items %] items = £[% pounds(base_price) %]
    4. [% END %]
  4. Please ensure you have read the bulky waste collection page on the council’s website
  5. In order to help us with your collection, we will ask you to optionally add pictures of the items to be collected and the location
  6. Before confirming your booking, check all the info provided is correct
    7. [% IF cobrand.moniker == 'peterborough' -%]
  7. You can amend the items in your booking up until [% c.cobrand.bulky_nice_cancellation_cutoff_time %] the day before the collection is scheduled by calling 01733 747474 (9am to 5pm Monday to Friday excluding bank holidays)
    8. [% END %] [% UNLESS cobrand.call_hook('bulky_enabled_staff_only') %] [% IF cobrand.moniker == 'peterborough' %]
  8. Cancellations made by [% c.cobrand.bulky_nice_collection_time %] the day before collection is scheduled are entitled to a refund
    9. [% END %] [% END %] [% IF cobrand.moniker == 'kingston' || cobrand.moniker == 'sutton' %]
  9. Bookings are final and non refundable
    10. [% END %]
